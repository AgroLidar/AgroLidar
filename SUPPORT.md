# Support

## Getting Help

For usage and integration questions, use one of the following channels:

- **GitHub Discussions** for how-to questions and architectural guidance.
- **GitHub Issues** for reproducible bugs and feature requests.
- **Security reports** must follow [SECURITY.md](SECURITY.md) and be sent privately.

When opening a support request, include:

- AgroLidar version (`VERSION` and commit SHA)
- Python/Node versions
- Platform profile and relevant config files
- Exact command run and complete error output
- Steps to reproduce

## Triage Expectations

- Initial maintainer response target: **5 business days**.
- Safety/security-impacting issues are prioritized ahead of general support requests.
