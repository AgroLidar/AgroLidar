# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.9.0] - 2026-03-27
### Added
- Added safety gate with configurable policy.
- Added pre-commit hooks and branch protection.
- Added MLflow experiment tracking.
- Added FastAPI inference server with health/metrics/predict endpoints.
- Added ONNX export and validation pipeline.
- Added hard case mining and review queue.
- Added BEV data schema and synthetic data generator.
- Added CI pipeline (lint, test, evaluate, smoke-train).

## [0.1.0] - Initial pipeline setup
### Added
- train/evaluate/compare/promote scripts.
- Dockerfile, Makefile, and GitHub Actions.
