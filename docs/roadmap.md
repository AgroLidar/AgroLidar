# Roadmap

This roadmap captures current engineering direction and release readiness priorities.

## Near-term (0.10.x)

- Expand benchmark scenarios for dense vegetation and multi-actor scenes.
- Increase inference server observability (structured logs + metrics surface).
- Strengthen configuration validation for platform-specific constraints.
- Improve model calibration and dangerous-class recall stability.

## Mid-term (0.11.x)

- Add reproducible dataset versioning metadata in retrain outputs.
- Introduce deployment profiles for edge accelerators.
- Expand integration tests for promotion and rollback workflows.

## 1.0.0 release readiness

- Stable API and model artifact contract
- Hardened operational documentation for enterprise deployments
- Formal release and deprecation policy
- Extended regulatory and safety assurance evidence packaging

## Non-goals (current phase)

- Online production self-training
- Unreviewed autonomous model promotion
- Hardware-specific optimization promises without validated benchmarks
